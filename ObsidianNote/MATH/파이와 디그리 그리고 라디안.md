워주율 PI는 원에둘레를 지름으로 나눈값이다.

 따라서.
 PI = l / 2r
 이라고 표현할수 있다

이때 반지름의길이를 1이라고 하면
PI = l / 2 이 이된다
즉 PI 값은 반원의 둘레에 값과 같다

이때 라디안 
반원에 둘레 radian 은 = PI radian = 180디그리 (180도)
가 된다

따라서 PI radian = 180도 이다
그러므로 1 radian = 180 / PI 값이 된다 대충 180 / 3.14... = 57.2958 이다

따라서,
디그리 => 라디안 : PI / 180 을 하고
라디안 => 디그리 : 180 / PI 을 하면 된다
